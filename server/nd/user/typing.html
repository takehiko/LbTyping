• Adjust the size of your browser so that the entire top frame is visible.
  • Three seconds after pressing the "Start" button, a program code appears in the frame. If you type correctly, you will advance to the next character. When you type until the end, you are finished.
  • The text is case sensitive.
  • You don't have to type spaces or line breaks. They will be ignored if typed.
  • There is no feature for auto-completion of parentheses. Type in the order shown.
