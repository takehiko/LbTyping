Replay Selection

task:

Select a replay.There are two types of replays: your replay and the hand model's replay.
If the number of simultaneous replays is 1, you can select one of the two types of replays.
If the number of simultaneous replays is 2, you can choose one from your own replays and one from the hand model's replays.
Note that the "Replay" button cannot be pressed if there is no replay data for either of them.
If you press the "Replay" button after making a selection, the replay screen will be displayed.
